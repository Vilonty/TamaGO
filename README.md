<h1>Тамагочи</h1>
Объяснение версий
<p>a - alpha</p>
<p>b - beta</p>
<p>o - original (готовая вресия)</p>
<br>
<p>Пример: a1.4.2</p>
<p>Тут a - alpha, 1 - это номер балванка, 4 - это порядковый номер созданной программы, 2 - это порядковый номер изменений в этой программе</p>
